# HealthSupport

## サイト概要

健康を気遣う人のためのサポートツール

### サイトテーマ

あなたの毎日をちょっと健康に

### テーマを選んだ理由

健康維持のために運動やダイエットを頑張りたいけど、一人ではなかなかモチベーションが続かず途中でやめてしまう。私自身、そんな経験を何度もしてきました。そこで、同じように頑張る仲間と繋がり、励ましあえる場所があったらいいなと思い、このテーマを選びました。

### ターゲットユーザ

健康維持や、ダイエット・筋トレを頑張りたい人

### 主な利用シーン

日々の健康管理、運動やダイエットのモチベーション維持に
